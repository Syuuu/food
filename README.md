# 小曦的厨房 · 手机端点餐页

一个纯前端的手机友好型点餐页面，列出你会做的菜，让女朋友随时挑选；点一下就能分享点餐内容（含微信场景）。

## 使用方式
1. 打开 `index.html`，直接在手机浏览器或任意静态空间访问即可。
2. 她选定菜品后，点“点餐”按钮，会调用系统分享（手机会出现包括微信在内的分享入口）；若设备不支持分享，会生成一段可复制的文本自行发送，微信内会自动复制并提示点右上角分享。

## 功能
- 预置菜单 20 道菜，配套菜名生成的图片展示，并按荤菜 / 素菜 / 粉面 / 汤羹 / 小吃 / 西式分类筛选。
- 手机端自适应的亮色餐馆风格 UI，选择菜品后直接“点餐”分享，无需填写额外信息。
- 通过浏览器分享点餐内容，提供复制备用的文本（微信内自动复制并提示使用分享菜单）。

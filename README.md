# 小曦的厨房 · 手机端点餐页

一个纯前端的手机友好型点餐页面，列出你会做的菜，让女朋友每天选一次；提交后会自动打开邮件窗口，提醒你备菜。

## 使用方式
1. 打开 `index.html`，在 `script.js` 中把 `KITCHEN_EMAIL` 改成你自己的收件邮箱。
2. 把项目部署到任意静态空间（或直接在手机浏览器打开本地文件）。
3. 她选定菜品、填写备注并提交后，会触发系统邮件，正文也会显示在下方方便复制。

## 功能
- 预置菜单 20 道菜，附带 AI 生成的图片展示。
- 手机端自适应的餐馆风格 UI，可选择菜品并填写日期、昵称、备注。
- 本地存储今日提交状态，默认每天仅能提交一次。
- 一键生成 `mailto:` 邮件，自动带上主题与正文。

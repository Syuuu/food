# 小曦的厨房 · 手机端点餐页

一个纯前端的手机友好型点餐页面，列出你会做的菜，让女朋友每天选一次；提交后可用浏览器分享点餐内容（含微信场景）。

## 使用方式
1. 打开 `index.html`，直接在手机浏览器或任意静态空间访问即可。
2. 她选定菜品并提交后，会调用系统分享（手机会出现包括微信在内的分享入口）；若设备不支持分享，会生成一段可复制的文本自行发送，微信内会自动复制并提示点右上角分享。

## 功能
- 预置菜单 20 道菜，配套菜名生成的图片展示。
- 手机端自适应的餐馆风格 UI，可选择菜品后直接分享，无需填写日期或备注。
- 本地存储今日提交状态，默认每天仅能提交一次。
- 通过浏览器分享点餐内容，提供复制备用的文本。
